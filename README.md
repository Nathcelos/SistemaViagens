# Sistema de Gerenciamento de Viagens

Projeto desenvolvido em Java para gerenciamento de pacotes de viagens e cadastro de vendas.

## 📋 Funcionalidades

O sistema permite:

### Cadastro do pacote de viagem
- Tipo de transporte
- Valor do transporte em dólar
- Descrição da hospedagem
- Valor da diária da hospedagem
- Destino da viagem
- Quantidade de dias
- Margem de lucro (%)
- Taxas adicionais em dólar

### Cálculos realizados
- Total da hospedagem
- Total do pacote de viagem

### Cadastro da venda
- Nome do cliente
- Forma de pagamento
- Cotação do dólar em reais

### Informações exibidas na venda
- Nome do cliente
- Forma de pagamento
- Total do pacote em dólar
- Total do pacote em reais
